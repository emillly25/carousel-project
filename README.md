# carousel-project

### 프로젝트 실행 방법

npm run dev

### 프로젝트 구조

```
├── index.html
├── style.css
├── main.js
└── public
    ├── assets
    ├── images
```

### 주요 기능

-   이전, 다음 버튼을 통한 슬라이더 이동
-   슬라이더 자동 재생 (간격: 2000ms - default)
-   재생 속도 조절 가능 (1000ms ~ 5000ms)
-   스와이프 제스쳐로 슬라이더 이동
-   인디케이터
